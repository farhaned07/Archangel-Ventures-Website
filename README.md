# Archangel — Company Website

Public website for **Archangel**, an operating company building AI systems, trade infrastructure, and private access networks across Southeast Asia and South Asia.

This repository contains the front-end implementation for the company website and brand surface.

## What this demonstrates

- Company and product positioning for an AI operating company
- Clean, executive-grade web interface design
- Next.js implementation with modern React and TypeScript
- Structured content for multiple business lines
- Production-minded front-end setup for deployment on Vercel

## Company structure represented

```text
Archangel Intelligence  → AI systems, audits, workflow automation
Archangel Trade         → Thailand setup, BOI, visas, market entry
Archangel Private Office → Bangladesh ↔ Thailand access and response network
```

## Stack

```text
Next.js 16
React 19
TypeScript
Tailwind CSS 4
Framer Motion
Lucide React
ESLint
```

## Running locally

```bash
npm install
npm run dev
```

Open the local development server at:

```text
http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```
